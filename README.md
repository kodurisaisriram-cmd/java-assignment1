# java-assignment1
Mini Banking System – Java
Overview

The Mini Banking System is a menu-driven Java application that simulates basic banking operations. It allows users to create accounts, deposit and withdraw money, search for accounts, and display account details. The system maintains a minimum balance condition and stores all account records using file handling to ensure data persistence.

Features

Create new bank account

Deposit money

Withdraw money with minimum balance validation

Search account by account number

Display all accounts

File-based data storage

Technologies Used

Java

Object-Oriented Programming (OOP)

File Handling (BufferedReader, BufferedWriter)

Exception Handling
